# Казкар Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build a full-stack Ukrainian/English fairy tale generator (Казкар) that streams stories and illustrates them via local Ollama, with persistent archive, edit/delete, theme + locale toggles, matching the visual design in `docs/_ Standalone.html`.

**Architecture:** Spring Boot 4 + WebFlux backend (port 8080) calls Ollama via WebClient and persists to Postgres via JPA (sync wrapped in `Mono.fromCallable(...).subscribeOn(boundedElastic())`). React 19 + Vite frontend (port 5173) consumes SSE for streaming, polls for illustration readiness. All under one repo `kazka/`.

**Tech Stack:** Java 25, Spring Boot 4, Gradle 9 Kotlin DSL, Spring Data JPA + Hibernate, Postgres 16 + Flyway, Spring WebFlux + WebClient, JUnit 5 + WebTestClient + Testcontainers + WireMock; React 19 + TypeScript + Vite, React Router v7, Vitest + React Testing Library, no UI framework. Ollama runs on host with `gemma3:4b` and `x/flux2-klein`.

**Spec reference:** `docs/superpowers/specs/2026-04-27-kazkar-design.md`

---

## Working agreements

- Run all backend commands from `backend/`. Run all frontend commands from `frontend/`.
- Commit after each task completes (the final step of every task is a commit).
- Use Conventional Commit prefixes (`feat:`, `chore:`, `test:`, `fix:`, `docs:`).
- All UI strings are looked up via `t('key')` — never hardcoded — so EN translations remain in sync.
- All Ollama URLs come from `OllamaProperties` — never hardcoded.
- All blocking JPA calls inside `@Service` methods called from WebFlux must be wrapped in `Mono.fromCallable(...).subscribeOn(Schedulers.boundedElastic())`.

```java
package com.kazka.ollama;

import com.fasterxml.jackson.databind.JsonNode;
import com.fasterxml.jackson.databind.ObjectMapper;
import org.springframework.http.MediaType;
import org.springframework.stereotype.Component;
import org.springframework.web.reactive.function.client.WebClient;
import reactor.core.publisher.Flux;
import reactor.core.publisher.Mono;

import java.util.Map;

@Component
public class OllamaClient {
    private final WebClient wc;
    private final OllamaProperties props;
    private final ObjectMapper mapper = new ObjectMapper();

    public OllamaClient(WebClient ollamaWebClient, OllamaProperties props) {
        this.wc = ollamaWebClient;
        this.props = props;
    }

    public Flux<String> streamGenerate(String model, String prompt) {
        return wc.post()
            .uri("/api/generate")
            .contentType(MediaType.APPLICATION_JSON)
            .bodyValue(Map.of("model", model, "prompt", prompt, "stream", true))
            .retrieve()
            .bodyToFlux(String.class)
            .flatMap(line -> {
                try {
                    JsonNode n = mapper.readTree(line);
                    String resp = n.path("response").asText("");
                    boolean done = n.path("done").asBoolean(false);
                    if (resp.isEmpty() && done) return Flux.empty();
                    return Flux.just(resp);
                } catch (Exception e) {
                    return Flux.error(e);
                }
            });
    }

    public Mono<String> generateImage(String model, String prompt) {
        return wc.post()
            .uri("/api/generate")
            .contentType(MediaType.APPLICATION_JSON)
            .bodyValue(Map.of("model", model, "prompt", prompt, "stream", false))
            .retrieve()
            .bodyToMono(JsonNode.class)
            .flatMap(n -> {
                JsonNode images = n.path("images");
                if (!images.isArray() || images.isEmpty()) {
                    return Mono.error(new RuntimeException("No image in response"));
                }
                return Mono.just(images.get(0).asText());
            });
    }

    public Mono<Void> pullModel(String name) {
        return wc.post()
            .uri("/api/pull")
```

```java
package com.kazka.story;

import com.kazka.story.dto.GenerationRequest;
import org.springframework.core.io.ClassPathResource;
import org.springframework.stereotype.Component;

import java.nio.charset.StandardCharsets;
import java.util.List;
import java.util.Map;

@Component
public class PromptBuilder {
    private final String systemUk = load("prompts/system-uk.txt");
    private final String systemEn = load("prompts/system-en.txt");
    private final String illusStyle = load("prompts/illustration-style.txt");

    private static final Map<String,Integer> TARGET_WORDS = Map.of(
        "short", 150, "medium", 400, "long", 800);

    public String buildStoryPrompt(GenerationRequest req) {
        boolean uk = "uk".equals(req.language());
        String system = uk ? systemUk : systemEn;
        int target = TARGET_WORDS.getOrDefault(req.length(), 400);
        String chars = String.join(uk ? ", " : ", ", req.characters());

        String userPrompt = uk
            ? "Тема казки: %s.\nГоловні персонажі: %s.\nВік дитини: %s років.\nДовжина: приблизно %d слів."
                .formatted(req.theme(), chars, req.ageGroup(), target)
            : "Story theme: %s.\nMain characters: %s.\nChild's age: %s years.\nLength: approximately %d words."
                .formatted(req.theme(), chars, req.ageGroup(), target);

        return system + "\n\n" + userPrompt;
    }

    public String buildIllustrationPrompt(String title, List<String> characters) {
        return title + " featuring " + String.join(", ", characters) + ". " + illusStyle;
    }

    private String load(String path) {
        try (var in = new ClassPathResource(path).getInputStream()) {
            return new String(in.readAllBytes(), StandardCharsets.UTF_8).trim();
        } catch (Exception e) {
            throw new IllegalStateException("Cannot load " + path, e);
        }
    }
}
```

```java
package com.kazka.illustration;

import com.kazka.config.UploadsProperties;
import org.springframework.stereotype.Service;

import java.nio.file.*;
import java.util.Base64;
import java.util.UUID;

@Service
public class ImageStorageService {
    private final Path dir;

    public ImageStorageService(UploadsProperties props) {
        this.dir = Path.of(props.dir()).toAbsolutePath();
        try { Files.createDirectories(dir); }
        catch (Exception e) { throw new IllegalStateException(e); }
    }

    public String savePng(UUID id, String base64) throws Exception {
        byte[] bytes = Base64.getDecoder().decode(base64);
        Path target = dir.resolve(id + ".png");
        Files.write(target, bytes, StandardOpenOption.CREATE, StandardOpenOption.TRUNCATE_EXISTING);
        return "/uploads/" + id + ".png";
    }

    public void delete(UUID id) {
        try { Files.deleteIfExists(dir.resolve(id + ".png")); }
        catch (Exception ignored) {}
    }
}
```

```java
package com.kazka.story;

import com.kazka.ollama.OllamaClient;
import com.kazka.story.dto.*;
import jakarta.persistence.EntityNotFoundException;
import org.springframework.data.domain.*;
import org.springframework.stereotype.Service;
import reactor.core.publisher.Flux;
import reactor.core.publisher.Mono;
import reactor.core.scheduler.Schedulers;

import java.util.*;

@Service
public class StoryService {
    private final OllamaClient ollama;
    private final StoryRepository repo;
    private final PromptBuilder prompts;

    public StoryService(OllamaClient ollama, StoryRepository repo, PromptBuilder prompts) {
        this.ollama = ollama;
        this.repo = repo;
        this.prompts = prompts;
    }

    public Flux<SseEvent> streamGenerate(GenerationRequest req) {
        return Mono.fromCallable(() -> createPending(req))
            .subscribeOn(Schedulers.boundedElastic())
            .flatMapMany(story -> {
                StringBuilder buf = new StringBuilder();
                String prompt = prompts.buildStoryPrompt(req);
                return Flux.<SseEvent>concat(
                    Flux.just(SseEvent.meta(Map.of("id", story.getId().toString()))),
                    ollama.streamGenerate(ollama.props().textModel(), prompt)
                        .doOnNext(buf::append)
                        .map(SseEvent::token),
                    Mono.fromCallable(() -> {
                        String full = buf.toString().trim();
                        String title = extractTitle(full);
                        String body  = extractBody(full);
                        story.setTitle(title);
                        story.setContent(body);
                        repo.save(story);
                        return SseEvent.done(Map.of(
                            "id", story.getId().toString(),
                            "title", title));
                    }).subscribeOn(Schedulers.boundedElastic())
                ).onErrorResume(err -> Flux.just(SseEvent.error(err.getMessage())));
            });
    }

    public Mono<PageResponse<StoryDto>> list(int page, int size) {
        return Mono.fromCallable(() ->
                repo.findAllByOrderByCreatedAtDesc(PageRequest.of(page, Math.min(size, 100))))
            .subscribeOn(Schedulers.boundedElastic())
            .map(p -> PageResponse.of(p, StoryDto::from));
    }

    public Mono<StoryDto> get(UUID id) {
        return Mono.fromCallable(() -> repo.findById(id)
                .orElseThrow(() -> new EntityNotFoundException("Story " + id + " not found")))
            .subscribeOn(Schedulers.boundedElastic())
            .map(StoryDto::from);
    }

    public Mono<StoryDto> update(UUID id, UpdateStoryRequest req) {
        return Mono.fromCallable(() -> {
            Story s = repo.findById(id)
                .orElseThrow(() -> new EntityNotFoundException("Story " + id + " not found"));
            s.setTitle(req.title());
            s.setContent(req.content());
            return StoryDto.from(repo.save(s));
        }).subscribeOn(Schedulers.boundedElastic());
    }

    public Mono<Void> delete(UUID id) {
        return Mono.<Void>fromRunnable(() -> {
            if (!repo.existsById(id)) throw new EntityNotFoundException("Story " + id + " not found");
            repo.deleteById(id);
        }).subscribeOn(Schedulers.boundedElastic());
    }

    private Story createPending(GenerationRequest req) {
        Story s = new Story();
        s.setId(UUID.randomUUID());
        s.setTitle(""); // filled in on done
        s.setTheme(req.theme());
        s.setCharacters(req.characters());
        s.setAgeGroup(req.ageGroup());
        s.setLength(req.length());
        s.setLanguage(req.language());
        s.setContent("");
        s.setIllustrationStatus(IllustrationStatus.PENDING);
        return repo.save(s);
    }

    static String extractTitle(String full) {
        int nl = full.indexOf('\n');
        String first = (nl < 0 ? full : full.substring(0, nl)).trim();
        return first.isEmpty() ? "Без назви" : first;
    }

    static String extractBody(String full) {
        int nl = full.indexOf('\n');
        return nl < 0 ? "" : full.substring(nl + 1).trim();
    }
}
```

```java
    @PostMapping(value = "/generate", produces = MediaType.TEXT_EVENT_STREAM_VALUE)
    public Flux<ServerSentEvent<Object>> generate(@Valid @RequestBody GenerationRequest req) {

                .build());
    }

    @PostMapping("/{id}/illustrate")
    public Mono<ResponseEntity<Void>> illustrate(@PathVariable UUID id) {
        illustrations.illustrate(id)
            .subscribeOn(Schedulers.boundedElastic())
            .subscribe();
        return Mono.just(ResponseEntity.accepted().build());
    }

    @GetMapping
    public Mono<PageResponse<StoryDto>> list(
        @RequestParam(defaultValue = "0") int page,
        @RequestParam(defaultValue = "20") int size) {
        return stories.list(page, size);
    }

    @GetMapping("/{id}")
    public Mono<StoryDto> get(@PathVariable UUID id) { return stories.get(id); }

    @PutMapping("/{id}")
    public Mono<StoryDto> update(@PathVariable UUID id, @Valid @RequestBody UpdateStoryRequest req) {
        return stories.update(id, req);
    }

    @DeleteMapping("/{id}")
    public Mono<ResponseEntity<Void>> delete(@PathVariable UUID id) {
```

Note the mapping output. The `template.html` references each font by UUID inside `url("UUID")`. We'll list them all under two `@font-face` family blocks (Lora and Nunito) — Lora is files 0–18a, Nunito is files 19–35; but the manifest only has 19 files because the standalone bundles each subset+style+weight separately.

For our purposes: each `.woff2` is one (family, style, weight, unicode-range) combination. Easiest path: serve all 19 files under a small `fonts.css` that mirrors the `@font-face` blocks from `docs/extracted/template.html`, replacing each `url("UUID")` with `url("/fonts/asset_<i>_<uuid8>.woff2")`.

Use this script to generate `fonts.css` automatically:

```bash
node -e "
const fs=require('fs');
const tpl=fs.readFileSync('docs/extracted/template.html','utf8');
const keys=JSON.parse(fs.readFileSync('docs/extracted/manifest_keys.json'));
let css=tpl.match(/<style>([\s\S]*?)<\/style>/)[1]; // first style block contains @font-face
keys.forEach((k,i)=>{
  const short=k.uuid.slice(0,8);
  css=css.split('url(\"[font]\")').join('url(\"/fonts/asset_'+i+'_'+short+'.woff2\")');
});
// the placeholders were inserted by extract_bundle.mjs; if originals had real UUIDs, replace those too
keys.forEach((k,i)=>{
  const short=k.uuid.slice(0,8);
  css=css.split('url(\"'+k.uuid+'\")').join('url(\"/fonts/asset_'+i+'_'+short+'.woff2\")');
});
fs.mkdirSync('frontend/src/design',{recursive:true});
fs.writeFileSync('frontend/src/design/fonts.css',css);
console.log('Wrote fonts.css',css.length,'chars');
"
```

```tsx
import { createContext, useContext, useState, type ReactNode, useMemo, useEffect } from 'react';
import uk from '../locales/uk.json';
import en from '../locales/en.json';

export type Locale = 'uk' | 'en';
const dictionaries = { uk, en } as const;

type Ctx = {
  locale: Locale;
  setLocale: (l: Locale) => void;
  t: (key: string) => string;
};

const LocaleContext = createContext<Ctx | null>(null);
const STORAGE_KEY = 'kazka.locale';

function detectInitial(): Locale {
  if (typeof window === 'undefined') return 'uk';
  const stored = window.localStorage?.getItem(STORAGE_KEY) as Locale | null;
  if (stored === 'uk' || stored === 'en') return stored;
  return navigator.language?.startsWith('en') ? 'en' : 'uk';
}

function lookup(dict: unknown, path: string): string | undefined {
  return path.split('.').reduce<unknown>((acc, k) => {
    if (acc && typeof acc === 'object' && k in (acc as Record<string, unknown>)) {
      return (acc as Record<string, unknown>)[k];
    }
    return undefined;
  }, dict) as string | undefined;
}

export function LocaleProvider({ children }: { children: ReactNode }) {
  const [locale, setLocaleState] = useState<Locale>(detectInitial);

  useEffect(() => {
    document.documentElement.lang = locale;
    try { localStorage.setItem(STORAGE_KEY, locale); } catch {}
  }, [locale]);

  const value = useMemo<Ctx>(() => ({
    locale,
    setLocale: setLocaleState,
    t: (key) => {
      const v = lookup(dictionaries[locale], key)
              ?? lookup(dictionaries.uk, key);
      if (v === undefined) {
        if (import.meta.env.DEV) console.warn('[i18n] missing key', key);
        return key;
      }
      return v;
    },
  }), [locale]);

  return <LocaleContext.Provider value={value}>{children}</LocaleContext.Provider>;
}

export function useLocale() {
  const ctx = useContext(LocaleContext);
  if (!ctx) throw new Error('useLocale must be used within LocaleProvider');
  return ctx;
}
```

```ts
import type { GenerationRequest } from './types';

export interface SseHandlers {
  onMeta?: (data: { id: string }) => void;
  onToken?: (data: { text: string }) => void;
  onDone?: (data: { id: string; title: string }) => void;
  onError?: (data: { message: string }) => void;
}

export async function streamGeneration(req: GenerationRequest, handlers: SseHandlers): Promise<void> {
  let res: Response;
  try {
    res = await fetch('/api/stories/generate', {
      method: 'POST',
      headers: { 'content-type': 'application/json', 'accept': 'text/event-stream' },
      body: JSON.stringify(req),
    });
  } catch (e) {
    handlers.onError?.({ message: (e as Error).message });
    return;
  }
  if (!res.ok || !res.body) {
    handlers.onError?.({ message: `HTTP ${res.status}` });
    return;
  }
  const reader = res.body.getReader();
  const dec = new TextDecoder();
  let buf = '';
  while (true) {
    const { done, value } = await reader.read();
    if (done) break;
    buf += dec.decode(value, { stream: true });
    let idx;
    while ((idx = buf.indexOf('\n\n')) !== -1) {
      const block = buf.slice(0, idx);
      buf = buf.slice(idx + 2);
      dispatch(block, handlers);
    }
  }
}

function dispatch(block: string, h: SseHandlers) {
  let event = 'message';
  let dataRaw = '';
  for (const line of block.split('\n')) {
    if (line.startsWith('event:')) event = line.slice(6).trim();
    else if (line.startsWith('data:')) dataRaw += (dataRaw ? '\n' : '') + line.slice(5).trim();
  }
  if (!dataRaw) return;
  try {
    const data = JSON.parse(dataRaw);
    switch (event) {
      case 'meta':  h.onMeta?.(data); break;
      case 'token': h.onToken?.(data); break;
      case 'done':  h.onDone?.(data); break;
      case 'error': h.onError?.(data); break;
    }
  } catch (e) {
    h.onError?.({ message: 'Failed to parse SSE data: ' + (e as Error).message });
  }
}
```

```tsx
import { useState, type KeyboardEvent } from 'react';
import './TagInput.css';

interface Props {
  value: string[];
  onChange: (next: string[]) => void;
  placeholder?: string;
  max?: number;
}

export function TagInput({ value, onChange, placeholder, max = 6 }: Props) {
  const [draft, setDraft] = useState('');
  function commit() {
    const trimmed = draft.trim();
    if (!trimmed || value.includes(trimmed) || value.length >= max) {
      setDraft('');
      return;
    }
    onChange([...value, trimmed]);
    setDraft('');
  }
  function onKey(e: KeyboardEvent<HTMLInputElement>) {
    if (e.key === 'Enter' || e.key === ',') { e.preventDefault(); commit(); }
    if (e.key === 'Backspace' && !draft && value.length) {
      onChange(value.slice(0, -1));
    }
  }
  return (
    <div className="tag-input">
      {value.map(tag => (
        <span key={tag} className="tag">
          {tag}
          <button type="button" aria-label={`Remove ${tag}`} onClick={() => onChange(value.filter(t => t !== tag))}>×</button>
        </span>
      ))}
      <input
        value={draft}
        onChange={e => setDraft(e.target.value)}
        onKeyDown={onKey}
        onBlur={commit}
        placeholder={value.length === 0 ? placeholder : ''}
      />
    </div>
  );
}
```

```tsx
import './StoryStream.css';

interface Props { title: string; text: string }

export function StoryStream({ title, text }: Props) {
  // Title is the first line of `text` until `done` event arrives, then
  // parent passes the title separately and the body excludes it.
  const display = title ? text.replace(new RegExp(`^${escapeRegex(title)}\n?`), '').trim() : text;
  return (
    <article className="story-stream">
      {title && <h2 className="story-title">{title}</h2>}
      <div className="story-body">
        {display.split(/\n+/).map((para, i) => <p key={i}>{para}</p>)}
      </div>
    </article>
  );
}

function escapeRegex(s: string) { return s.replace(/[.*+?^${}()|[\]\\]/g, '\\$&'); }
```

```tsx
import { useEffect, useState } from 'react';
import { api } from '../../lib/apiClient';
import { useLocale } from '../../lib/localeContext';
import type { Story } from '../../lib/types';
import { PlaceholderSvg } from './PlaceholderSvg';
import './IllustrationFrame.css';

interface Props { storyId: string; initial: Story }

const POLL_MS = 2000;
const TIMEOUT_MS = 60_000;

export function IllustrationFrame({ storyId, initial }: Props) {
  const { t } = useLocale();
  const [story, setStory] = useState(initial);

  useEffect(() => {
    if (story.illustrationStatus !== 'PENDING') return;
    const start = Date.now();
    let cancelled = false;
    let timer: ReturnType<typeof setTimeout>;

    async function tick() {
      if (cancelled) return;
      try {
        const next = await api.getStory(storyId);
        if (cancelled) return;
        setStory(next);
        if (next.illustrationStatus === 'PENDING' && Date.now() - start < TIMEOUT_MS) {
          timer = setTimeout(tick, POLL_MS);
        }
      } catch {
        if (cancelled) return;
        timer = setTimeout(tick, POLL_MS);
      }
    }
    timer = setTimeout(tick, POLL_MS);
    return () => { cancelled = true; clearTimeout(timer); };
  }, [storyId, story.illustrationStatus]);

  if (story.illustrationStatus === 'READY' && story.illustrationPath) {
    return (
      <div className="illustration-frame">
        <img src={story.illustrationPath} alt={story.title} />
      </div>
    );
  }
  if (story.illustrationStatus === 'FAILED') {
    return (
      <div className="illustration-frame illustration-frame--failed">
        <PlaceholderSvg />
      </div>
    );
  }
  return (
    <div className="illustration-frame illustration-frame--loading">
      <PlaceholderSvg />
      <p>{t('story.illustrationLoading')}</p>
    </div>
  );
}
```

```tsx
  const [submitting, setSubmitting] = useState(false);
  const [streamId, setStreamId] = useState<string | null>(null);
  const [streamText, setStreamText] = useState('');
  const [streamTitle, setStreamTitle] = useState('');
  const [doneStory, setDoneStory] = useState<Story | null>(null);

    setError(null);
    setStreamId(null); setStreamText(''); setStreamTitle(''); setDoneStory(null);

      onMeta: m => setStreamId(m.id),
      onToken: tk => setStreamText(prev => prev + tk.text),
      onDone: async d => {
        setStreamTitle(d.title);
        setSubmitting(false);
        api.illustrate(d.id).catch(() => {});

          const s = await api.getStory(d.id);
          setDoneStory(s);
        } catch {}
      },

        setError(e.message || t('errors.ollamaUnreachable'));
        setSubmitting(false);
      },
    });
  }

  return (
    <>
      <Hero rightSlot={<StoryForm submitting={submitting} onSubmit={onSubmit} />} />

      {error && <div role="alert" className="error-toast">{t('errors.ollamaUnreachable')}</div>}

          <StoryStream title={streamTitle} text={streamText} />
```

```tsx
export function StoryDetailPage() {
  const { id } = useParams<{ id: string }>();
  const navigate = useNavigate();
  const { t } = useLocale();
  const [story, setStory] = useState<Story | null>(null);
  const [editing, setEditing] = useState(false);
  const [draftTitle, setDraftTitle] = useState('');

  const [confirmDel, setConfirmDel] = useState(false);

        onEdit={() => setEditing(true)}

        </article>
```

````markdown
Local-AI Ukrainian/English fairy tale generator. Streams personalized bedtime stories via Ollama and illustrates them with a local image model.
````
