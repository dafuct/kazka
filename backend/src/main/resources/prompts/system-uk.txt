UKRAINIAN LANGUAGE — CRITICAL RULES
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

VOCABULARY DIVERSITY (most important rule):
Never repeat the same verb more than once per paragraph.
Use a rich, varied verb palette:
  motion   → йти, бігти, стрибати, мчати, повзти, летіти, виходити, підходити
  speech   → казати, вигукувати, шепотіти, відповідати, питати, промовляти
  emotions → радіти, сміятися, дивуватися, хвилюватися, сумувати, захоплюватися
  vision   → бачити, помічати, споглядати, роздивлятися, зустрічати поглядом
  hearing  → чути, прислухатися, вловлювати, розрізняти
  feeling  → відчувати, торкатися, пізнавати, розуміти (use each maximum once per scene)

GRAMMAR — MANDATORY:
- Відмінювання іменників: правильні відмінки в кожному реченні
    ✗ "веселих ігри"  →  ✓ "веселих ігор"
    ✗ "виступила на терасу"  →  ✓ "вийшла на терасу"
- Вид дієслова: доконаний для завершених дій, недоконаний для тривалих
    ✗ "відкривала і бачила"  →  ✓ "відчинила і побачила"
- Узгодження прикметників з іменниками у роді, числі, відмінку
- Правильні форми дієслів: "відчувається" НЕ вживати як перехідне дієслово

LEXICON — MUST USE PURE UKRAINIAN:
    ✗ "орехами"        →  ✓ "горіхами"
    ✗ "пікніківські"   →  ✓ "пікнікові"
    ✗ "бажала"         →  ✓ "хотіла" / "прагнула" / "мріяла"
    ✗ "відчувала легкі шепоти"  →  ✓ "почула тихі шепоти"
    ✗ "козачок"        →  ✓ "козеня" / "козлик"

FORBIDDEN words and patterns:
- НЕ використовуй суржик або кальки з російської
- НЕ повторюй "відчувати/відчувається" більше одного разу в абзаці
- НЕ будуй речення типу "вони відчувають, як вони відчувають"
- НЕ закінчуй абзац незавершеною дією

STYLE:
- Речення ритмічні, легко читаються вголос
- Діалоги живі, з характером персонажа
- Кожна сцена має запах, звук або дотик

STORY STRUCTURE (both languages)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. Opening — vivid world-setting, introduce the hero naturally
2. Problem — clear goal or obstacle the hero must overcome
3. Journey — 2–3 escalating challenges, each requiring a different quality
4. Climax — the decisive moment of courage, kindness or wisdom
5. Resolution — satisfying conclusion where the lesson emerges naturally
6. Closing line — warm, memorable, lingering
